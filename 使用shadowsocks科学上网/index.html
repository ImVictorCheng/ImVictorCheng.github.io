使用shadowsocks科学上网

Posted on 2016-12-08 |

考虑到好多同学问我关于科学上网的问题，所以干脆写到这里好了。 （不知道什么是科学上网的朋友就不用科学上网了……）

之前试过很多科学上网的工具，但是大多不稳定，而且速度慢。目前来看最好用的还是shadowsocks（中文名：影梭，有时简写为ss）。在最后会放一些其他的工具，作为备用。

软件

官网、GitHub主页、windows版本，或者从这里下载3.3.6版（windows）。 也可以pip安装，如果你特别想这样干的话：

pip install shadowsocks

打开就能用。

打开后是这个样子： 左侧是你的所有节点，右侧是节点服务器的信息。

安卓版。 上不了play的到GitHub主页或这里下载apk。（个人觉得科学上网软件只提供play链接不提供apk都很蠢……） 用法很简单，可以扫描二维码或者粘贴配置信息代码来添加节点。

节点

节点是使用shadowsocks科学上网的关键。可以自己租（或者和别人合租）海外的服务器搭建节点（没试过……），也可以买提供商的节点（据说会快一点，流量多一点，稳定性好一点？），也可以用许多网站提供的免费节点。

在搜索引擎上直接搜“shadowsocks免费节点”一般就可以找得到。也可以到g+找shadowsocks的社区，有很多人经常在那里分享提供免费节点的网站。比方说这个：shadowsocks免费帐号。大部分这些免费节点网站都可以每天签到送流量，节点速度有快有慢，一半访问网页是够的，看YouTube的话有的能到720，有的只能看360。

有个fb上的小组，里面这方面的大神不少，如果想自己租服务器可以去问；也有很多人发一些合租的广告（没试过，不知道靠不靠谱）；还有一些其他相关的东西。

PAC

Proxy auto-config，代理自动配置，可以帮你选择哪些网站使用代理，哪些不用。（像什么知乎微博之类的当然希望它不要用代理啦，又慢又浪费流量。）

右键点击ss的任务栏图标，有一项叫系统代理模式，里面可以选择PAC模式或者全局模式。PAC模式就是仅PAC文件里面列出的网站走代理；全局模式就是所有网站都走代理。

还有一个选项叫PAC，里面有一个选项叫编辑本地PAC文件…，打开后是一个txt文件，里面列出了走代理的网站。可以在这里按照格式（我也没研究过这个格式是什么意思……）补充别的网站，可以注释掉其实能访问的网站（比方说在校园网下可以注释掉google）。平时上网的时候遇到一些访问特别慢的国外网站，可以添加到这个文件里，经常能显著提高访问速度。

让UWP应用走ss代理

用WindowsModernAppsTools，下载后打开，选择小应用网络设置选项，选择需要走代理的app，点击右下角的允许选中项目使用本地代理。

其它科学上网工具

ss很稳定，很够用，但以防万一还是要收集一些备用的。 常见的有赛风、蓝灯（蓝灯的GitHub，现在好像要收费）、VPN Gate（VPN Gate已经被墙，但可以找一个不常用的邮箱地址订阅它的镜像，它每天会给你发三次，帝国主义真是亡我之心不死）。
