win10增强工具

Posted on 2016-11-13 | | | Visitors:

本文持续更新

介绍几个我自己搜集到的小工具，或者隐藏的使用方法。

自定义快捷键

win10自带了不少快捷键，除了Ctrl+C这些最最最基础的操作系统快捷键以外，还有一些windows独有的，不太广为流传的快捷键，大多涉及到windows徽标键（win键），可以直接参考这个网页，这里简单列举一下我常用的： - win+R打开“运行”窗口 - win+I打开Windows设置（非控制面板） - win+A打开操作中心 - win+S打开小娜 - win+D显示桌面/恢复桌面 - win+G打开游戏录制工具栏 这应该是一个经常被忽略的工具。 这个工具栏本意是让你录制游戏视频或者截图，但其实可以在任何程序下运行，所以可以用来： + 窗口截屏，这同PrtSc键的区别是只截取当前窗口内的内容，而不包括任务栏等等，例： + 录制操作，比方说你想远程教别人怎么安装mathematica，可以录制一下你的操作过程。（不过这个我没试过，只是听说可以实现。） + 视频截取，这个一目了然，不过注意全屏和隐藏鼠标指针。 图片或者视频自动保存在此电脑-视频-捕获这个文件夹里。如果你想更换默认文件夹，只需把Captures文件夹剪切到目标位置就好了。 - win+L锁定。如果你的电脑上没有专门的锁定按钮的话，这个很方便。 - Alt+Tab切换窗口。如果你还不知道，或者你只是见过别人做过这么酷的事情却不知道怎么做到的话（比方说以前的我）。按下这个快捷键后是可以用上下左右键或者鼠标来选择窗口的。 - win+Tab任务视图，同上一个的区别是松开键盘不会消失。可以用上下左右键和鼠标选择。在win10中还可以选择和创建虚拟桌面。 - win+Ctrl+D创建新的虚拟桌面。 - win+Ctrl+方向键切换虚拟桌面。 - win+方向键分屏和最大最小化。 - Alt+F4关闭窗口，所有窗口都关闭后会打开关机选项。（我所知的最快捷的关机方法） - Ctrl+Alt+Del广为流传的打开任务管理器的快捷键，但在win10还需要一步才能进入任务管理器，所以我一般用右键任务栏的方法找任务管理器。 - 在窗口内按Alt可以选择菜单栏，不太常用。 - Alt+上在任务管理器可以向上。

注意到，win10中有一些快捷键是重复的，比方说win+Q和win+S；还有些键的组合没有定义，比方说win+Y；还有些快捷键几乎用不到，比方说win+U打开轻松访问中心。我们可以把这些组合键改成自定义的快捷键。使用AutoHotKey这个软件，在官网正常下载安装。

安装完成之后，直接在桌面新建一个AutoHotKey Script文件，右键选择Edit Script，正常会用记事本打开，然后就可以配置了。AutoHotKey的功能很强大，详细可以参考在线的帮助文件，目前我只用了运行程序的功能。

#f::
    run, F:\Program Files\Everything\Everything.exe
return

#代表win键，#f表示win+F，后面两个冒号表示这个是快捷键。（与AutoHotKey的缩写功能区分。）可以在一个脚本里写好几个快捷键，接着写就行了。

把一些常用的软件直接设成快捷键，可以解放任务栏。

写好脚本后，可以设置自启动，将将该脚本生成一个“快捷方式”，然后将此快捷方式放置到程序自启动文件夹之下即可。自启文件夹一般是

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp

本地搜索

windows自己的资源管理器的搜索功能实在是太太太差劲了……也不知道为什么。下面介绍一个软件叫Everything。

1.4.1Beta版已经支持简体中文了，所以可以直接下载最新版。

搜索框支持正则表达式。如果希望在某一特定文件夹下搜索，直接把路径粘贴在搜索框内，然后空格搜索即可。

Everything不支持搜索文件内容。

由于Everything每次启动都需要花一定时间建立索引，所以建议设置开机自启：直接打开任务管理器，选择“启动”选项卡，把Everything的状态改为启用。同时建议用上面的AutoHotKey设置一个搜索快捷键，就可以完美代替资源管理器的搜索功能了。

透明任务栏

StartIsBack。这个软件的主要功能是恢复win8以前的开始菜单，不过我还是很喜欢win10带metro磁贴的开始菜单的，所以只用它的透明任务栏功能。

在配置的“外观”选项卡，勾选使用自定义任务栏颜色，把右边的不透明度调到0。点击“切换”选项卡，把第一项改成“显示开始屏幕”，把第二项改成什么也不做，即可。
